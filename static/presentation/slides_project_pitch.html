👨‍💻 Macroeconomic forecasting with ML 📊¶

  • The forecasting macroeconomic conditions is of fundamental importance for households, businesses and governments
  • Traditional macro-econometric forecasting models involves discretionary choices about data and models (Smalter Hall, 2018)
  • Forecasts based on expert domain judgement (e.g. surveys) might be based on wrong intuitions
  • Can we use machine learning techniques to predict macroeonomic variables?

Pros

  • ML techniques let the data speak
  • ML techniques better capture non-linearities in the data
  • The available literature says that forecasts from different ML models generally perform better than traditional models

Cons

  • Models are not transparent ('black box')
  • It is important to have a theory of how the economy works!
  • ML models work better on large numbers of data (only few data available in macro)

Studies forecasting inflation using ML¶

  • Frade Moro (2022) uses deep learning (LSTM) to forecast inflation for Brazil using several features (e.g. BRL-USD exchange-rate, base interest rate)
  • Kohlscheen (2021) examines the drivers of inflation for 20 advanced economies using random forests
  • Other studies have investigated the performance of different ML models on the forecast of other macroeconomic variables (unemployment, gdp growth)

Project: Inflation forecasting with deep learning¶

  • Why the US? Data are consistent and easily available (hopefully!)
  • What do we do?
    • Obtain data using the dbnomics API (or other online sources...)
    • Identify the right features using random forests (see Kohlscheen 2021)
    • Forecast inflation with deep learning (RNN LSTM) (see Frade Moro 2022)
  • The final output will be an "inflation dashboard" (frontend) including an API
